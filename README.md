# Delivery-performance-optimization-python
End-to-end Supply Chain Analytics project analyzing 172K+ e-commerce orders to identify delivery delays, measure profitability, detect operational bottlenecks, and predict late deliveries using Machine Learning. Built with Python, Pandas, Matplotlib, Scikit-learn, and Random Forest with SMOTE.
<h1 align="center">🚚 Supply Chain Analytics & Late Delivery Risk Prediction</h1>

<p align="center">
An end-to-end Data Analytics and Machine Learning project that analyzes supply chain performance, identifies the root causes of delivery delays, measures their financial impact, and predicts late delivery risk using Python.
</p>

<hr>

<h2>📌 Overview</h2>

<p>
This project analyzes <b>172,765 e-commerce orders</b> (2015–2018) to evaluate delivery performance, profitability, and operational efficiency. The workflow includes data preprocessing, business KPI analysis, exploratory data analysis (EDA), root cause analysis, bottleneck detection, time series analysis, and a Machine Learning model for predicting late deliveries.
</p>

<h2>🎯 Key Features</h2>

<ul>

  <li>🔍 Root Cause Analysis</li>
  <li>⚙️ Bottleneck Detection</li>
  <li>📈 Time Series Analysis</li>
  <li>🤖 Machine Learning Modeling</li>

</ul>
